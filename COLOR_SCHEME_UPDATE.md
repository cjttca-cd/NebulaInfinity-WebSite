# 色彩方案更新 - Linear.app 风格

网站配色已完全更新为 Linear.app 风格，提供更专业、更易读的视觉体验。

## 🔄 主要变化

### 1. 移除玻璃拟态效果

**之前:**
```css
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(10px);
```

**现在:**
```css
background: var(--surface); /* #1a1b1f */
border: 1px solid var(--border); /* #27272a */
```

### 2. 实心背景替代透明背景

所有卡片、模态框、导航栏现在使用实心背景，提供：
- ✅ 更好的可读性
- ✅ 更清晰的层次结构
- ✅ 更专业的外观
- ✅ 更好的性能（无 blur 滤镜）

### 3. 高对比度文字

文字颜色从半透明改为实心高对比度颜色：
- 主要文字更清晰 (`#f6f8fa`)
- 次要文字更易读 (`#b4b4b4`)
- 层次分明但不刺眼

### 4. Linear 风格的紫蓝色系

从青色 (#2DD4BF) 改为专业的紫蓝色 (#5e6ad2)：
- 更专业、更企业化
- 与 Web3/科技主题完美契合
- 与 Linear.app 品牌一致

## 🎯 设计原则

### Linear.app 的设计哲学

1. **清晰度优先** - 高对比度，易读性强
2. **专业外观** - 实心表面，明确边界
3. **微妙精致** - 边框和阴影恰到好处
4. **性能优化** - 无复杂滤镜和效果

### 实现细节

#### 颜色层次
```
背景层级:
#0d0e11 (最深) → #16171d → #1a1b1f → #1f2024 → #25262b (最亮)

文字层级:
#f6f8fa (最亮) → #b4b4b4 → #71717a → #52525b (最暗)

边框层级:
#27272a (默认) → #3f3f46 (悬停) → #52525b (焦点)
```

#### 强调色使用
- 主按钮: `#5e6ad2`
- 链接悬停: `#7c3aed`
- 渐变: `#5e6ad2 → #8b5cf6`

## 📱 响应式和可访问性

### 对比度比率

所有文字颜色符合 WCAG AA 标准：

| 组合 | 对比度 | 等级 |
|------|--------|------|
| `#f6f8fa` on `#0d0e11` | 15.8:1 | AAA ✅ |
| `#b4b4b4` on `#0d0e11` | 9.2:1 | AAA ✅ |
| `#71717a` on `#0d0e11` | 5.1:1 | AA ✅ |

### 暗色模式优化

当前配色本身就是暗色模式，针对长时间使用优化：
- 不使用纯黑避免眼睛疲劳
- 深紫灰背景更柔和
- 文字对比度高但不刺眼

## 🚀 使用新配色

### CSS 变量

所有颜色通过 CSS 变量定义在 `styles/theme.css`:

```css
/* 背景 */
var(--background)           /* #0d0e11 */
var(--background-secondary) /* #16171d */
var(--surface)              /* #1a1b1f */
var(--surface-hover)        /* #1f2024 */

/* 文字 */
var(--text-primary)         /* #f6f8fa */
var(--text-secondary)       /* #b4b4b4 */
var(--text-tertiary)        /* #71717a */

/* 强调色 */
var(--accent-primary)       /* #5e6ad2 */
var(--accent-primary-hover) /* #7c3aed */

/* 边框 */
var(--border)               /* #27272a */
var(--border-hover)         /* #3f3f46 */
```

### 渐变

```css
var(--gradient-primary)     /* #5e6ad2 → #8b5cf6 */
var(--gradient-secondary)   /* #8b5cf6 → #7c3aed */
```
